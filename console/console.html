En la cosola se puede hacer codigo javascript secuencialmente, paso por paso

----------Funciones de registro

-assert()- aparece un mensaje de error en la consola si la afirmacion es falsa. Si la afirmacion es verdadera no aparecera nada (NO ESTANDAR)

-clear()- la consola se limpia, esto sirve para cuando tengo muchos errores en la consola puedo limpiarlo

-error ()- Muestra un mensaje de error en la consola web

-Info ()- Emite un mensaje informativo a la consola web. En firebox y chorme, se muestra un peque√±o icono "i" junto a estos elementos en el registro de la consola web

-log()-muetra un mensaje en la consola web (o del interprete javascript)

-table () - Esta fincion tomoa un argumento obligatorio: data, que debe ser un array o un objeto, y un paramentro adicional: columns y nos muestra una tabla en consola

- warn () - imprime un mensaje de advertencia en la consola web

-dir () - despliega una lista interactiva del las propiedades del objeto de javascript especificado.[no estandar]

-------Funcion de conteo

-count () Registra el numero de veces que se llama a count(), esta funcion toma con argumento opcional una etiqueta-

-resetCount() Resetea el contador console.count()

-------Funciones de agrupamiento

group () crea un nuevo grupo en linea en el registro de la consola web

groupEnd() remueve un grupo en linea en el registro de la consola web

groupCollapsed() crea un grupo en linea pero contraido, el usuario debe abrirlo para verlo

-----Funciones de temporizador

-time() inicia el temporizador

-timeEnd() detiene el temporizador

-timeLog() registra el tiempo actual que lleva a ejecutar un funcion
